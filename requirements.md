# 要件定義書：PMS栄養辞典

## 1. プロジェクト概要

- **背景**：看護師資格と分子栄養学・腸もみ・美筋ヨガの専門知識を活かしながら、育児と並行してPMSサポート事業を運営している。限られた時間の中でも一人ひとりのクライアントにしっかり向き合いたいという思いがある一方、セッションのたびに同じ栄養情報を説明する手間や、参加者が自習できるツールがないという課題を感じていた。
- **目的**：育児と仕事を両立しながらも質を落とさないサポートを実現するため、不調から逆引きで栄養素・食材にアクセスできる辞典アプリを構築する。まー自身の業務時短と、プログラム参加者の自己学習を同時に叶えるツールとして機能させる。
- **スコープ（含む）**：不調→栄養素→食材の3階層閲覧、MVP版はログインなし
- **スコープ（含まない）**：食事記録、症状トラッキング、ショッピング連携（v2以降）

---

## 2. ユーザーストーリー

```
As a PMSプログラム参加者,
I want 自分の不調（例：むくみ）をクリックして関連栄養素を調べられること,
So that カウンセリング外でも自分のペースで知識をつけられる.

As a 育児をしながらPMSサポート事業を運営するまー,
I want クライアントにURLを共有するだけで栄養情報を届けられること,
So that 子どもとの時間を大切にしながらも、手を抜かずにクライアントをサポートできる.
```

## 4. 非機能要件

- **パフォーマンス**：ページ読み込み3秒以内（スマホ回線でも快適に）
- **セキュリティ**：MVP期はURL秘匿のみ、v1.0でEmail + Google OAuth導入
- **可用性**：Vercelの無料枠SLAに準拠
- **スケーラビリティ**：当初〜50名程度のプログラム参加者を想定
- **デザイン**：ソフトピンク×オフホワイト、スマホ最適化（参加者がLINEから開く想定）

## 5. 制約条件

- **予算**：無料枠で完結（Vercel + Supabase free tier）
- **期間**：1〜2週間（MVP）
- **技術**：Next.js + Supabase + Vercel（予定）

### 不調と栄養素の対応理由

#### S-01 むくみ
- **カリウム**：細胞内外の水分バランスを調整し、余分なナトリウムを排出することでむくみを軽減する
- **マグネシウム**：血管の緊張をゆるめ、血流を改善することで水分の滞りを防ぐ
- **ビタミンB6**：女性ホルモンの代謝を助け、黄体期のホルモン変動によるむくみを緩和する
- **タンパク質**：血中アルブミンの材料となり、血管内の浸透圧を保つことで水分が組織に漏れ出るのを防ぐ

#### S-02 頭痛・片頭痛
- **マグネシウム**：脳血管の過剰な収縮・拡張を抑え、片頭痛の発症頻度を下げることが研究で示されている
- **ビタミンB2**：ミトコンドリアのエネルギー産生を支え、脳細胞のエネルギー不足による頭痛を予防する
- **鉄**：鉄不足による脳への酸素供給不足が頭痛・片頭痛を悪化させるため、特に月経のある女性に重要
- **ビタミンE**：血小板の凝集を抑制し、脳血管の血流を改善することで片頭痛の予防に関与する
- **オメガ3脂肪酸**：プロスタグランジン（炎症物質）の産生を抑え、血管の炎症による頭痛を緩和する。月経前の頭痛にも関連が示されている

#### S-03 イライラ・気分の波
- **マグネシウム**：神経の興奮を抑制し、ストレスホルモン（コルチゾール）の過剰分泌を抑えることで気分を安定させる
- **ビタミンB6**：セロトニン・ドーパミン合成の補酵素として働き、気分の安定に直接関与する
- **亜鉛**：神経伝達物質の合成と脳の神経可塑性を支え、不足するとうつ・不安感が増しやすい
- **鉄**：ドーパミン・セロトニンの合成に必要で、不足すると感情コントロールが難しくなる

#### S-04 疲労感・だるさ
- **鉄**：ヘモグロビンの材料として酸素を全身に運ぶ。不足すると酸素供給が低下し、慢性的なだるさの最大原因となる
- **ビタミンB12**：赤血球の生成と神経機能の維持に必要で、不足すると貧血・神経疲労が起こる
- **ビタミンC**：副腎でのコルチゾール産生を支え、ストレス下での疲労回復を促進する。鉄の吸収率も高める

#### S-05 肌荒れ・ニキビ
- **亜鉛**：皮膚の細胞再生・コラーゲン合成・皮脂コントロールに関与し、ニキビの炎症を抑制する
- **ビタミンA**：皮膚の角化を正常化し、毛穴詰まりやターンオーバーの乱れを改善する
- **ビタミンC**：コラーゲン合成の補酵素として肌のハリを保ち、抗酸化作用で炎症による肌ダメージを軽減する
- **オメガ3脂肪酸**：抗炎症作用により肌の赤みや炎症性ニキビを抑え、皮膚バリア機能を強化する
- **タンパク質**：皮膚・爪・髪の主成分であるコラーゲンやケラチンの材料。不足すると肌の再生が遅れ、荒れやすくなる
- **ビタミンE**：強力な抗酸化作用で皮脂の酸化を防ぎ、肌の炎症を抑える。ビタミンCと組み合わせることで抗酸化効果がさらに高まる

#### S-06 便秘・お腹の張り
- **マグネシウム**：腸管に水分を引き込み便を軟化させ、蠕動運動を促進することで自然な排便をサポートする
- **食物繊維**：腸内細菌のエサとなり善玉菌を増やすとともに、便のかさを増して排便リズムを整える
- **ビタミンB1**：腸の自律神経の働きを支え、蠕動運動の低下による便秘・腹部膨満を改善する

#### S-07 眠れない・睡眠の乱れ
- **マグネシウム**：神経の興奮を鎮めGABA（抑制系神経伝達物質）の働きを高め、入眠しやすい状態をつくる
- **ビタミンB6**：睡眠ホルモン（メラトニン）の前駆体であるセロトニン合成を助け、睡眠リズムの調整に関与する
- **タンパク質**：睡眠ホルモン（メラトニン）の材料となるアミノ酸（トリプトファン）はタンパク質から摂取されるため、不足は睡眠の質低下に直結する

### 栄養素詳細データ構造（MVP：データは開発中に入力）

各栄養素は以下の構造で定義する。

```typescript
type Nutrient = {
  id: string           // 例: "magnesium"
  name: string         // 例: "マグネシウム"
  category: string     // 例: "ミネラル"
  description: string  // 働き（テキスト）
  foods: string[]      // 豊富な食材リスト
  dailyIntake: string  // 摂取目安量（例: "女性 270mg/日"）
  relatedSymptoms: string[] // 関連する不調IDリスト（例: ["S-01", "S-03"]）
}
```
